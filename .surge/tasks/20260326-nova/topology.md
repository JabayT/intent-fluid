## 任务拓扑报告

| 项目 | 值 |
|------|-----|
| **任务类型** | 串行（Serial） |
| **估计迭代轮数** | 2-3 轮 |
| **配置** | max_iterations=5 / parallel_agent_limit=10 |

### 串行主链
1. **世界观与创新概念构建**（medium）— 确定子类型、X-Factor、核心设定
2. **章节大纲与叙事结构**（medium）— 5+章节分配、节奏安排
3. **角色设计与关系图谱**（small）— 核心角色原型与动机
4. **最终整合输出**（medium）— 中文交付，Master Concept + Outline + Characters

> 全部串行，无需并行节点（各模块强依赖上游设定）

### Phase Role Config

| Phase | Customized Role |
|-------|-----------------|
| analyze | 资深科幻文学策划编辑，精通类型文学市场分析与叙事结构理论，擅长识别商业潜力与创新点的平衡 |
| research | 科幻文学与流行文化研究专家，熟悉全球科幻出版趋势、读者偏好数据及前沿科学概念，能快速定位未被充分开发的创意空间 |
| design | 科幻世界架构师，擅长构建自洽的世界观体系与多线叙事框架，精通"第一性原理"式创新概念的落地设计 |
| implement | 中文科幻写作工程师，具备严谨的中文文学表达能力，能将设计文档转化为商业级质量的小说大纲、角色档案与世界设定文档 |
| qa | 科幻叙事质量评审专家，从市场吸引力、创新度、结构连贯性、角色深度四个维度评估交付物，熟悉反俗套检测方法论 |
| retro | 创意项目回顾专家，善于提炼创作流程中的经验教训与可复用模式 |
